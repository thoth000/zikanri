# Zikanri

<img width="30%" src="./images/zikanri_shaped.png">

アプリ甲子園2020年大会に応募するアプリ。

Flutterを用いたクロスプラットフォームアプリで
iOSとAndroidに両対応している。

時間を価値アリ・価値ナシに分けて記録することができ
毎日を効率的に振り返ることができる。
